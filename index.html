Landmarked

An online travel journal to track all the great places you visit and all the trips you go on. Integrated with Google Maps API for interactive map, and AWS for user uploaded image hosting.

Funday

A clone of 'monday.com', Funday is a fully dynamic task-management web application where users can update and maintain groups of task lists among teams. Funday is a single page, fullstack web application built with Ruby and Ruby on Rails/AJAX for backend, React/Redux for frontend.
